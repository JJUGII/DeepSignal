# 투자공격성 다이얼 (1~10) — 기획서 (구현 전)

> 목표: **설정 하나(1~10)** 로 엔진 전체의 공격성을 조절. 숫자가 클수록 **리스크가 커지는 대신, 맞았을 때 수익 폭도 커진다.**
> 핵심: "더 크게 잃을 각오 = 더 크게 먹을 구조". 단순히 자주 사는 게 아니라 **수익 증폭 레버**가 같이 켜진다.

---

## 1. 한 개의 손잡이로 작동하는 구조

```
DEEPSIGNAL_AGGRESSION = N (1~10)   ← 사용자가 정하는 단 하나의 값
        │
        ▼
  resolve_aggression(N)  ← N을 받아 모든 세부 파라미터를 계산하는 '해석기'
        │
        ├─ 어떤 전략을 켤지 (코인/주식/레버리지/인버스)
        ├─ 포지션 크기 배수
        ├─ 레버리지 배율 (1x→2x→3x)
        ├─ 진입 임계값 (낮을수록 더 자주 진입)
        ├─ 익절 방식 (고정 익절 → "수익 달릴 때 끝까지" 트레일링)
        ├─ 손절·일일손실 한도 (높을수록 더 버팀)
        └─ 매매 주기 (느림 → 빠름)
        │
        ▼
  모든 러너(코인·주식·레버리지·고속회전)가 이 값을 읽어 동일하게 동작
```

사용자는 **숫자 하나만** 바꾸면 됨. 대시보드에 슬라이더로 노출 가능.

---

## 2. 단계별 매핑 (리스크 ↑ = 수익 잠재력 ↑)

| 단계 | 켜지는 것 | 포지션 배수 | 레버리지 | 진입 문턱 | 익절 방식 | 일일 손실한도 | 예상 MDD |
|---|---|---|---|---|---|---|---|
| **1 (현재)** | 추세추종 ETF만 | 0.3x | 1x | 매우 높음 | 추세이탈 시 | 매우 작음 | ~-10% |
| **2** | +검증 엣지 소액 | 0.4x | 1x | 높음 | 고정 익절 | 작음 | ~-15% |
| **3** | 코인 무승인 자동 | 0.6x | 1x | 높음 | 고정 TP/SL | 작음 | ~-20% |
| **4** | +국내주식 자동 | 0.8x | 1x | 표준 | 동적 TP/SL | 표준 | ~-25% |
| **5** | +해외·패스트레인 | 1.0x | 1x | 표준 | 동적+부분익절 | 표준 | ~-30% |
| **6** | +피라미딩·레짐공격 | 1.3x | 1.5x | 약간 낮음 | **트레일링(달리기)** | 큼 | ~-38% |
| **7** | 캡 상향·임계 하향 | 1.6x | 2x | 낮음 | 트레일링 넓게 | 큼 | ~-48% |
| **8** | **레버리지 ETF 주력** | 1.8x | 2x | 낮음 | 트레일링+피라미딩 | 매우 큼 | ~-58% |
| **9** | 3x 레버리지 | 2.0x | 3x | 매우 낮음 | 끝까지 달리기 | 거의 무제한 | ~-72% |
| **10** | 풀투입·안전최소 | 2.0x(전액) | 3x | 최저(거의 다 진입) | 손절만, 익절무제한 | 사실상 해제 | -80%+ / 청산 |

> 핵심 단서: 6단계부터 **익절을 "고정"에서 "트레일링(수익이 계속 나면 안 팔고 따라간다)"으로 전환** → 큰 추세에서 수익 폭이 폭발적으로 커지는 구조. 이게 "리스크만큼 수익도 커진다"의 핵심 장치.

---

## 3. "수익 폭"을 키우는 4가지 증폭 레버 (단계가 올라갈수록 강해짐)

1. **레버리지 배율**: 같은 자본에 2~3배 노출 → 맞으면 2~3배 수익 (틀리면 2~3배 손실)
2. **포지션 크기 배수**: 한 종목에 거는 비중 ↑
3. **트레일링 익절(let winners run)**: 고정 익절 5%로 끊지 않고, 추세가 살아있는 한 계속 보유 → +30%, +50%도 먹음
4. **피라미딩(불리기)**: 수익 중인 포지션에 추가매수 → 이긴 자리에 더 크게 베팅

→ 1~5단계는 주로 "빈도·종목수"만 늘지만, **6단계부터 위 4개가 본격 가동**되어 수익 상방이 열린다.

---

## 4. 단계가 내려도 지켜지는 안전 바닥 (전 단계 공통)
- **전역 킬스위치**는 어느 단계든 즉시 모든 신규매수 중단
- **EDGE_GATE 검증**: 8단계까지는 "검증된 전략만" 거래 (레버리지도 EDGE_GATE 통과해야)
- **9~10단계만** EDGE_GATE 완화 옵션 (= 검증 안 된 것도 베팅, 진짜 도박 구간 → 명시적 경고)
- 트레일링/하드손절은 레버리지 단계일수록 **더 좁게** 자동 설정(이미 P1에 구현됨)

---

## 5. 구현 시 작업 범위 (참고)
- `deepsignal/risk/aggression.py` 신규: `resolve_aggression(N)` → 파라미터 dict
- 각 러너(코인·주식·레버리지·intraday)가 시작 시 이 값을 읽어 적용
- 익절 모드 "트레일링 전환" 로직 (dynamic_tpsl + trailing_stop 연결)
- 대시보드 슬라이더 UI (1~10) + 현재 단계·예상 MDD 표시
- 단계 변경 시 텔레그램 통보 + 확인

---

## 6. 권장 운용 방식
- **종이→소액→증액**: 새 단계로 올리기 전 며칠 관찰
- 한 번에 2단계 이상 점프 금지 (리스크 급증)
- 8단계 이상은 **잃어도 되는 돈**으로만
- 추천 시작: 현재 1 → 검증되며 **4~5단계**까지가 "공격적이지만 합리적" 구간
