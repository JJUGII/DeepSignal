# DeepSignal 동작 원리 가이드

> 처음 보는 사람도 전체 구조를 이해할 수 있도록 작성한 설명서입니다.

---

## 한 줄 요약

**DeepSignal은 주식·코인 시장을 24시간 모니터링하면서, "지금 사도 될까? 팔아야 할까?"를 점수로 계산하고 자동으로 주문까지 내주는 시스템입니다.**

---

## 전체 흐름 한눈에 보기

```
시장 데이터 수집
    ↓
실시간 분석 (62개 지표 계산)
    ↓
K-GSQS 점수화 (0~100점)
    ↓
신호 판단 (NOTIFY / AUTO / STRONG)
    ↓
포지션 사이징 (얼마나 살지)
    ↓
동적 TP/SL 계산 (얼마에 익절/손절할지)
    ↓
주문 실행 (사람 승인 후 자동 주문)
    ↓
체결 추적 & 성과 피드백
```

---

## 1단계 — 시장 데이터를 실시간으로 수집한다

DeepSignal은 세 종류의 시장을 동시에 감시합니다.

| 자산 | 어디서 가져오나 | 무엇을 보나 |
|------|--------------|-----------|
| **코인** (비트코인·이더리움 등) | Binance 거래소 실시간 WebSocket | 1초마다 체결가·거래량 |
| **국내주식** (삼성전자·SK하이닉스 등) | KIS(한국투자증권) Open API | 실시간 체결 + 호가창 |
| **해외주식** (NVDA·AAPL·TSLA 등) | KIS Open API 해외 스트림 | 미국장 실시간 체결 |

데이터가 들어오면 1분봉·5분봉·15분봉 단위로 집계합니다. 이 봉 데이터가 분석의 원재료입니다.

---

## 2단계 — 62개 지표로 종목을 분석한다

봉 데이터가 만들어지면 즉시 62개 기술 지표를 계산합니다.

### 어떤 지표를 보나?

| 지표 유형 | 예시 | 무엇을 알려주나 |
|---------|------|--------------|
| **이동평균** | MA5, MA20, MA60 | 현재 추세가 상승인지 하락인지 |
| **거래량** | 평소 대비 거래량 배수 | 지금 "관심"이 몰리는 종목인지 |
| **호가잔량** | 매수벽 vs 매도벽 | 지금 사려는 사람이 많은지, 팔려는 사람이 많은지 |
| **모멘텀** | 1분·5분·15분 수익률 | 최근 움직임이 얼마나 강한지 |
| **시장 상대강도** | KOSPI 대비 강도 | 시장 전체 흐름보다 더 강한 종목인지 |
| **리스크** | ATR%, 갭 | 변동성이 너무 크지 않은지 |

> 62개를 모두 한 번에 수동으로 보는 건 인간이 할 수 없습니다. DeepSignal이 이걸 1분마다 자동으로 계산합니다.

---

## 3단계 — K-GSQS 점수로 매수 가능성을 수치화한다

62개 지표를 6개 영역으로 묶어서 하나의 점수(0~100점)로 만듭니다. 이것이 **K-GSQS (Korean General Signal Quality Score)** 입니다.

### 점수 구성

```
K-GSQS 총점 (0~100점)
├── 추세 점수     (20%) — MA 정배열, VWAP 위치
├── 거래량 점수   (20%) — 평소 대비 거래량, 매수 비율
├── 호가 점수     (20%) — 매수/매도 잔량 비율, 스프레드
├── 모멘텀 점수   (20%) — 단기 수익률, 체결강도
├── 시장 점수     (10%) — 시장지수 대비 상대강도
└── 리스크 점수   (10%) — 변동성, 거래정지 여부
```

### 신호 등급

| 점수 | 등급 | 의미 |
|------|------|------|
| 88점 이상 | **STRONG** 🔴 | 매우 강한 매수 신호 — 1.5배 비중 투입 |
| 82~87점 | **AUTO** 🟠 | 자동매수 대상 — 1.25배 비중 투입 |
| 72~81점 | **NOTIFY** 🟡 | 관심 종목 알림 — 기본 비중 투입 |
| 72점 미만 | 진입 불가 ⚫ | 거래 안 함 |

### 실제 예시

> NVDA가 갑자기 급등 → 거래량 3배 폭발 + MA 정배열 + 매수벽 강함  
> → K-GSQS 91점 → **STRONG** 신호 → 자동으로 매수 계획 생성

---

## 4단계 — 얼마나 살지 계산한다 (포지션 사이징)

"살 종목"이 결정되면 "얼마어치 살지"를 자동 계산합니다.

### 계산 방식

```
기본 투입금 = 총 자산의 10%
최대 투입금 = 총 자산의 25%

실제 투입금 = 기본 투입금 × 신호 강도 팩터
  STRONG(88점+) → 기본의 1.5배
  AUTO(82점+)   → 기본의 1.25배
  NOTIFY(72점+) → 기본의 1.0배
```

### 예시

> 총 자산 1,000만원 / NVDA 신호 STRONG  
> → 기본 100만원 × 1.5 = **150만원** 어치 매수

최대 5개 종목까지 동시에 보유할 수 있습니다.

---

## 5단계 — 얼마에 팔지 미리 계산한다 (동적 TP/SL)

매수하기 전에 "익절가(TP)"와 "손절가(SL)"를 **자동으로** 계산합니다.  
이것이 **동적 TP/SL** 입니다.

### TP/SL이란?

- **TP (Take Profit, 익절)**: 이 가격에 오르면 수익 실현
- **SL (Stop Loss, 손절)**: 이 가격에 내리면 손실 제한

### 어떻게 계산하나?

고정된 숫자가 아니라 **시장 상황에 따라 매번 달라집니다.**

**① 종목 등급별 기준값**

| 등급 | 해당 종목 | 익절 목표 | 손절 기준 |
|------|---------|--------|--------|
| A등급 | BTC, ETH, 삼성전자 | 넉넉히 | 여유있게 |
| B등급 | 대형주, 주요 알트코인 | 중간 | 중간 |
| C등급 | 중소형주 | 보수적 | 빡빡하게 |
| D등급 | 3배 레버리지 ETF 등 | 매우 보수적 | 매우 빡빡하게 |

**② 현재 시장 상황에 따라 조정**

| 시장 상태 | 감지 기준 | 익절/손절 조정 |
|--------|---------|-------------|
| 📈 상승장 | 지수 +1% 이상 | 익절 목표 늘림 |
| 😐 횡보장 | 변동 없음 | 기본 설정 |
| 📉 하락장 | 지수 -1% 이하 | 손절 기준 강화 |
| 💥 폭락 | 지수 -5% 이하 | 신규 진입 차단 |

**③ ATR(변동폭) 기반 자동 조정**

변동성이 높은 날은 TP/SL 폭을 넓히고,  
변동성이 낮은 날은 폭을 좁힙니다.

### 예시

> NVDA 주가 180달러에 매수  
> ATR = 3%(일일 평균 변동폭)  
> 현재 시장 = 상승장  
> → TP: +6% = 190.8달러 (익절)  
> → SL: -3% = 174.6달러 (손절)

## 6단계 — 자동으로 주문을 낸다

### 주문 전 안전장치 (4단계 승인 구조)

DeepSignal은 실수로 잘못된 주문이 나가지 않도록 여러 단계를 거칩니다.

```
1. 신호 감지
      ↓
2. 주문 계획 생성 (아직 실주문 아님)
      ↓
3. 자동 검증 (수량·가격·잔고 확인)
      ↓
4. 사람이 최종 승인 ← 이 단계까지는 실제 돈이 움직이지 않습니다
      ↓
5. KIS Open API로 실제 주문 전송
```

### 매수 주문

- **지정가(LIMIT) 매수**만 사용 — 현재가보다 약간 높은 가격에 주문
- 시장가 주문은 사용하지 않음 (슬리피지 방지)

### 매도 주문

매도도 자동화되어 있지만 반드시 **LIMIT(지정가) 주문**만 사용합니다.

```
매수 후 가격이 오르면 → TP 도달 → 자동으로 LIMIT SELL 주문
매수 후 가격이 내리면 → SL 도달 → 자동으로 LIMIT SELL 주문
```

---

## 7단계 — 결과를 학습해서 점점 똑똑해진다

DeepSignal은 매매가 끝난 후에도 계속 학습합니다.

### 자동 학습 사이클

```
신호 발생 → 진입가 기록
    ↓
5분·15분·30분 후 수익률 추적
    ↓
"이 신호가 맞았나?" 자동 평가
    ↓
K-GSQS 가중치 자동 최적화
(예: 거래량 지표가 더 잘 맞히면 거래량 비중 올림)
```

### 슬리피지 추적

- 내가 원했던 주문가격 vs 실제 체결가격 차이를 자동으로 기록
- 평균 슬리피지, 최대 슬리피지, P90 슬리피지를 분석해 주문 전략 개선

## 웹 대시보드에서 확인할 수 있는 것들

DeepSignal은 웹 브라우저에서 실시간으로 모든 것을 확인할 수 있습니다.

| 화면 | 볼 수 있는 것 |
|------|-------------|
| **암호화폐 탭** | 코인별 K-GSQS 점수판, 신호 이력, 현재 포지션 |
| **국내주식 탭** | 국장 실시간 점수, 보유 종목, 포지션 사이징, 계좌 대사 |
| **해외주식 탭** | 미국장 신호, 해외 포지션(달러→원화 환산), 슬리피지 통계 |
| **공통** | 거래 이력, 체결오차(슬리피지), 가중치 최적화 현황 |

## 자주 묻는 질문

**Q. 완전 자동으로 거래하나요?**  
A. 신호 감지·분석·계획 수립까지는 완전 자동입니다. 하지만 실제 돈이 움직이는 실전 주문은 반드시 사람이 승인해야 합니다. 이중·삼중 안전장치가 있어서 실수로 주문이 나가는 경우는 없습니다.

**Q. 손실이 나면 어떻게 되나요?**  
A. 매수할 때부터 손절가(SL)를 미리 계산해 둡니다. 가격이 SL에 도달하면 자동으로 매도 주문을 냅니다. 한 번에 잃을 수 있는 금액은 총 자산의 일부로 제한됩니다.

**Q. 모의 거래도 할 수 있나요?**  
A. 네. KIS 모의투자 계정과 연결하면 실제 돈 없이 똑같은 환경에서 테스트할 수 있습니다. 기본값이 모의투자 모드입니다.

**Q. 코인·주식·해외주식을 동시에 볼 수 있나요?**  
A. 네. 세 가지 시장을 동시에 분석하고, 하나의 웹 대시보드에서 통합해서 볼 수 있습니다.

**Q. 유니버스(감시 종목)는 어떻게 정하나요?**  
A. 매일 장 시작 전에 거래대금·거래량 상위 종목을 자동으로 선정합니다. 30분마다 목록을 갱신해서 핫한 종목을 놓치지 않습니다.

## 핵심 철학

> **"사람의 감을 대체하는 게 아니라, 사람이 놓칠 수 있는 것을 보완한다"**

DeepSignal은 62개 지표를 1분마다 확인하고, 30개 이상 종목을 동시에 추적하며, 밤새도록 미국 시장을 모니터링합니다. 이걸 사람이 직접 하기는 불가능하기 때문에 시스템이 대신합니다. 단, 최종 결정은 항상 사람이 내립니다.
